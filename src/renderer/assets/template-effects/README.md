Template Effect SVG Assets
==========================

Add curated SVG files to the effect folders in this directory. These folders match the template effect IDs used by the app:

- acid
- arcane
- cold
- darkness
- fire
- fog
- lightning
- nature
- poison
- psychic
- radiant
- storm
- thunder
- water
- web

Future renderer work can load these with `import.meta.glob` and choose one or more SVGs for a template effect.
