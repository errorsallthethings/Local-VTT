# Local VTT v0.1.13

This release focuses on release readiness, clearer issue intake, and regression coverage for file recovery and Player View scene switching.

## Highlights

- Added GitHub issue forms for bug reports and feature/enhancement work.
- Improved actionable error handling for metadata save failures and missing map or campaign assets.
- Added regression coverage for scene switching, Player View state transitions, and missing asset recovery.
- Added Windows-first installer upgrade and downgrade validation guidance.
- Hardened GitHub Actions release publishing with platform artifact checks and clearer immutable-release retry guidance.

## Release Notes

- Windows builds remain available as the `.exe` installer.
- macOS builds remain available as `.dmg` and `.zip` artifacts.
- Linux builds remain available as AppImage, Debian package, and RPM package artifacts.
- The Windows installer is not code signed, and macOS builds are not notarized yet.
