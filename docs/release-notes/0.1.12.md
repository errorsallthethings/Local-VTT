# Local VTT v0.1.12

This release adds Linux desktop build artifacts alongside the existing Windows and macOS packages.

## Added

- Added Linux AppImage builds.
- Added Linux Debian package (`.deb`) builds.
- Added Linux RPM package (`.rpm`) builds.
- Added Linux icon assets for desktop package integration.
- Updated the GitHub Actions release workflow to build and upload Linux artifacts.

## Verification

- `npm run check`
- `npm run build`

## Notes

- Windows builds remain available as the `.exe` installer.
- macOS builds remain available as `.dmg` and `.zip` artifacts.
- Linux users can choose `.AppImage`, `.deb`, or `.rpm` depending on their distribution and install preference.
- Windows builds are not code signed, and macOS builds are not notarized yet.
